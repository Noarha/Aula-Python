# Aula-Python
Código desenvolvido em aula 
O código solicita informações do cliente como nome, valor da compra e se possui desconto disponível na compra realizada. Foram apresentados três valores de compras como exemplo, no quais o valor de R$50 não teve oferta de desconto, R$ 150 foi ofertado R$15 de desconto e o último de R$ 300 recebeu R$30 de desconto. O código também foi escrito para repetição até que o cliente deseje encerrar a compra.
